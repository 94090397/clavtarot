# Changelog

All notable changes to ClavTarot will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-13

### Added
- Complete 78-card Rider-Waite tarot deck with upright and reversed meanings
- 22 Major Arcana cards (The Fool through The World)
- 56 Minor Arcana cards (Wands, Cups, Swords, Pentacles)
- 5 spread types: Single Card, Three Card, Celtic Cross, Love, Career
- AI-powered tarot card image generation via fal.ai
- Interactive CLI installer (`npx clavtarot@latest`)
- 3 built-in tarot reader personas (Mystica, Luna, Oracle) + custom option
- Daily fortune delivery via OpenClaw cron system
- Topic-specific readings (general, love, career, health)
- Cross-platform messaging support (Discord, Telegram, WhatsApp, Slack, Signal, Teams)
- Seeded daily fortune (same card all day for consistency)
- Complete SKILL.md with step-by-step instructions for OpenClaw agent
- Soul injection template for agent persona customization
- GitHub Sponsors and Buy Me a Coffee integration
